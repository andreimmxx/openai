# Эволюция (демо-версия)

Небольшая браузерная адаптация настольной игры **«Эволюция»** (по мотивам игры Дмитрия Кнорре) с упрощёнными правилами.

## Что реализовано

- 2 игрока с именами.
- Автоматическая генерация видов с разными свойствами.
- Кормовая база на раунд.
- Кормление травоядных и атака хищников.
- Подсчёт очков после завершения раунда.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 4173
```

После этого откройте `http://localhost:4173`.
